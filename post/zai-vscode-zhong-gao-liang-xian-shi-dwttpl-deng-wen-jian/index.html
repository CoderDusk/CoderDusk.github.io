在 VSCode 中高亮显示 dwt、tpl 等文件

2020-06-08 1 min read

.dwt / .tpl文件等模板文件虽然是 html 语法书写的，但 VSCode 默认也是无法将其按照 HTML 语法进行高亮显示的，得在设置中将这些文件格式和 html 进行关联。

搜索出来的很多文章展示的是老版本的设置方法和截图，如下：

新版 VSCode 的设置方法已经变了，新版软件中应该这样设置：

文件 -> 选项 -> 设置

在搜索框中输入关联，编辑相应的设置项

在配置文件里添加：

"files.associations": {
    "*.dwt": "html"
}

保存后就立即生效了。
